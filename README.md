# ATLAS - Adaptive Triage and Local Advisory System

Clinical decision support system for resource-limited healthcare settings.

## Features
- Offline-first Progressive Web Application
- AI-enhanced clinical decision support
- WHO SMART Guidelines implementation
- Multi-provider data synchronization

## Setup
```bash
npm install
npm run dev
```

## Environment Variables
Create `.env.local` with:
- `NEXT_PUBLIC_GEMINI_API_KEY`
- `NEXT_PUBLIC_API_URL`
